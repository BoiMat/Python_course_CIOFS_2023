# Lezione 2: Funzioni

**Argomenti Principali:**
- Sintassi.
- Definizione e Chiamata.
- Parametrizzazione di una funzione.
- Default values.
- Scope.
- Composizione di funzioni.
- Funzioni che ritornano valori.
- Funzioni ricorsive.
- Commenti e Docstrings.
